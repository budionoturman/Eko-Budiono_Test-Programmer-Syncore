TEST 1

  1. Mengapa setiap bahasa pemrograman itu terdapat framework, apa kegunaan framework ?

    jawaban : setiap bahasa pemograman memiliki framework karena adanya komunitas antar pengguna bahasa pemograman untuk saling berdiskusi dan memecahkan masalah bersama sehingga di dalam komunitas itu ada orang-orang yang bekerja sama membuat framework untuk memudahkan dan membatu programmer lain mengembangkan aplikasi.

  2. Jelaskan mengenai fungsi dan penggunaan if else, while, while do, switch case, for, foreach?

    jawaban :

      - if else digunakan untuk mengecek kondisi dari kondisi yang ditulis paling atas ke paling bawah secara berurut
      - while digunakan untuk melakukan perulangan terus menerus dan berhenti jika mencapai kondisi yang diinginkan
      - while do digunakan untuk melakukan perulangan sebanyak yang ditentukan tetapi dapat berhenti jika bertemu kondisi tertentu
      - switch case digunakan untuk mengecek pengkondisian yang sama berdasarkan case yang sudah ditentukan
      - for digunakan untuk melakukan perulangan sebanyak yang ditentukan di code
      - foreach digunakan untuk melakukan perulangan di file html sebanyak array yang ada

  3. Apa itu constructor, method, class, object, variabel, parameter, function, procedure ?

    Jawaban :

      -construktor dibuat untuk mengisi data parameter yang dibutuhkan
      -method dibuat untuk memisahkan tugas tugas sehingga satu tugas hanya satu method seperti getAll, getById
      -class dibuat untuk menyimpan dan memisahkan method method sesuai dengan karakteristik seperti User, Kabupaten, Kecamatan
      -variable dibuat/diinisialisasikan untuk menyimpan data
      -parameter terdapat pada dalam kurung method untuk mengirim data yang dibutuhkan oleh method
      function dalam bahasa PHP dapat digunakan untuk membuat method
      procedure adalah urutan atau langkah langkah sesuai dengan yang ditentukan

  4. Perbedaan method=”Post” dan “Get” ?

    jawaban : method post digunakan untuk mengirim data dari client ke server, method get digunakan untuk meminta data ke server oleh client

  5. Jelaskan konsep mengenai web service dan fungsi apa yang digunakan untuk memanggil web service client ?

    jawaban : web service merupakan layanan berbasis website dimana client dapat meminta request dan server mengirimkan response ke client

  6. Jelaskan mengenai MVC dan HMVC !

    jawaban : MVC(model, views, controller) model berhubungan dengan database, views berisi file yang menampilkan frontend, controller untuk menghubungan antara model dengan views
    HMVC(hierarki, model, views, controller) memisahkan model,views, controller sesuai dengan role user dengan memiliki mvc-nya sendiri

  7. Sebutkan fungi curl() dalam php!

    jawaban : curl(Client URL) library open source untuk membantu programmer php dalam mengembangkan aplikasi

  8. Bagaimana menampilkan sebuah element array dalam PHP ?

    jawaban : menampilkan element array di php dapat menggunakan perintah echo $namaVariableArray

  9. Perbedaan HTTP dan HTTPS ?

    jawaban : https memiliki tingkat keamanan yang lebih ditingkatkan daripada http

  10. Jelaskan Apa itu API? sebutkan contohnya?

    jawaban : API(Aplication Programming Interface) penghubung antara backend dan frontend untuk saling berbagi data, contoh api: PokeApi(pokemon), NewsApi(Berita)
